1️⃣ Accounts (authentification / gestion utilisateurs)
Méthode	Endpoint	Description	Permissions
GET	/users/	Liste des utilisateurs	Authenticated
POST	/users/	Créer un nouvel utilisateur	Admin uniquement
GET	/users/{id}/	Détails d’un utilisateur	Authenticated
PUT	/users/{id}/	Modifier un utilisateur	Admin uniquement
PATCH	/users/{id}/	Modifier partiellement un utilisateur	Admin uniquement
DELETE	/users/{id}/	Supprimer un utilisateur	Admin uniquement

4️⃣ File d’attente (FileAttente)
| Méthode | Endpoint                      | Description                      | Permissions          |
| ------- | ----------------------------- | -------------------------------- | -------------------- |
| GET     | /file_attente/                | Liste des clients en file        | Authenticated        |
| POST    | /file_attente/                | Ajouter un client à la file      | Receptionniste/Admin |
| GET     | /file_attente/{id}/           | Détails d’un client dans la file | Authenticated        |
| PUT     | /file_attente/{id}/           | Modifier client dans la file     | Receptionniste/Admin |
| PATCH   | /file_attente/{id}/           | Modifier partiellement           | Receptionniste/Admin |
| DELETE  | /file_attente/{id}/           | Supprimer de la file             | Admin uniquement     |
| POST    | /file_attente/{id}/commencer/ | Passer EN_ATTENTE → EN_COURS     | Coiffeur/Admin       |
| POST    | /file_attente/{id}/terminer/  | Passer EN_COURS → TERMINE        | Coiffeur/Admin       |
